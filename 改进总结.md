### 4. REST API 服务器 ✅

**新增文件：**
- `app/server.py` - FastAPI REST API + WebSocket 支持

**API 端点：**

| 端点 | 方法 | 说明 |
|------|------|------|
| `/health` | GET | 健康检查 |
| `/status` | GET | Agent 状态（上下文占用、活跃任务） |
| `/chat` | POST | 单次对话 |
| `/chat/stream` | POST | 流式对话（SSE） |
| `/run` | POST | 异步任务执行 |
| `/task/{id}` | GET | 查询任务状态 |
| `/ws` | WebSocket | 实时通信 |

**启动方式：**
```bash
# 直接启动
python -m app.server

# 或使用 uvicorn
uvicorn app.server:app --host 0.0.0.0 --port 8000 --reload

# 生产环境（多 worker）
gunicorn app.server:app -w 4 -k uvicorn.workers.UvicornWorker
```

**简历价值：**
- 展示 Web API 设计能力
- 体现异步编程功底（asyncio + WebSocket）
- 可部署为实际服务

## 项目亮点总结（简历用）

### 技术栈
- **核心：** Python 3.12+, asyncio, 类型注解
- **AI 集成：** Anthropic SDK, OpenAI 兼容 API
- **Web 框架：** FastAPI, WebSocket, SSE
- **容器化：** Docker, docker-compose
- **CI/CD：** GitHub Actions
- **测试：** pytest, 494 个测试用例

### 核心创新
1. **多智能体协作系统**
   - 5 个专业角色（Coordinator/Planner/Coder/Reviewer/Memorizer）
   - 分阶段编排引擎（orchestrate）
   - 自动 Agent 选择（Overlap Coefficient 算法）

2. **Wiki 式记忆系统**
   - digest → wiki 双层知识沉淀
   - 四域分类（对话/个人/Agent/任务）
   - 抗幻觉设计（检索优先级 wiki → digest → 不编造）

3. **技能自进化**
   - 成功任务自动结晶为 YAML 技能卡
   - 三维质量评分（清晰度/完整性/可操作性）
   - 使用衰减 + 自动淘汰机制

4. **上下文管理**
   - 三层渐进压缩（Prune → Summary → Emergency）
   - KV Cache 断点检测
   - Token 预算控制

### 架构设计
```
ai/          ← 核心类型定义（零依赖）
agent/       ← 20 个接口 + 核心逻辑
app/         ← CLI + REST API + 工具实现
```

### 代码质量
- 494 个测试，全部通过
- 接口先行（ABC/Protocol）
- 类型注解覆盖
- 分层单向依赖

## 环境配置清单

```bash
# 1. 克隆仓库
git clone https://github.com/Yang1999code/controllable-agent.git
cd controllable-agent

# 2. 创建虚拟环境
conda create -n controllable-agent python=3.12
conda activate controllable-agent

# 3. 安装依赖
pip install -e ".[all]"

# 4. 配置 API Key
cp .env.example .env
# 编辑 .env 填入你的 API Key

# 5. 运行测试
pytest tests/ -v

# 6. 启动 CLI
python -m app.cli

# 7. 启动 REST API
python -m app.server
# 访问 http://localhost:8000/docs
```
